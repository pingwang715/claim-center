# Claim Center

> Claim Center is a claims management system dedicated to insurance companies to manage claims online.

---

## 🚀 Features

- User registration and authentication
- Secure API integration
- Full life cycle of claims management
- Audit trail of claims and payments

---

## 🛠️ Tech Stack

**Frontend**
- React / TypeScript / HTML / CSS / Tailwind css

**Backend**
- Node.js / Java / Spring Boot

**Database**
- MySQL

**Other Tools**
- Docker
- Git
- CI/CD

---

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/pingwang715/claim-center.git

# Navigate into the project
